# Requirements
# Tested environment: Python 3.12.3, Windows 11, CPU-only (no GPU).
# Install with: pip install -r requirements.txt
#
# CLIP (OpenAI, for ViT-B/32 / ViT-L/14 vanilla checkpoints):
#   pip install git+https://github.com/openai/CLIP.git
#
# For fashion-domain CLIP (strongly recommended — directly addresses the
# "better than vanilla CLIP" bar in the assignment):
#   pip install open_clip_torch>=2.24.0
#   Then re-index with: python indexer/main.py --clip-model hf-hub:Marqo/marqo-fashionCLIP
#
# If groundingdino fails to initialize (weight download issue or CUDA mismatch),
# the detector module falls back automatically to OWL-ViT, which is pulled via
# the same transformers version listed here.
